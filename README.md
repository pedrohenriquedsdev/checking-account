# 💳 Sistema de Conta Corrente

Projeto desenvolvido em **C# com Programação Orientada a Objetos (POO)** para simular as operações básicas de uma **conta corrente**, aplicando conceitos fundamentais como classes, objetos, métodos, encapsulamento e separação de responsabilidades.

---

## 📌 Sobre o projeto

O objetivo deste projeto é praticar a construção de regras de negócio utilizando **POO em C#**, simulando o funcionamento de uma conta corrente por meio de um sistema em console.

O usuário pode interagir com um menu principal e realizar operações bancárias essenciais.

### ✅ Funcionalidades
- Criar conta corrente
- Consultar dados da conta
- Realizar depósito
- Realizar saque
- Utilizar limite de débito
- Menu interativo no console
- Validação de operações inválidas

---


## 📄 Responsabilidade dos arquivos

### `ContaCorrente.cs`
Responsável pelas **regras de negócio e dados da conta**.

Contém:
- ID da conta
- Saldo
- Limite de débito
- Método de depósito
- Método de saque
- Validações

### `TelaPrincipal.cs`
Responsável pela **interação com o usuário no console**.

Contém:
- Menu principal
- Navegação entre opções
- Entrada de dados
- Exibição de mensagens
- Visualização das informações da conta

### `Program.cs`
Responsável pelo **ponto de entrada da aplicação**.

Contém:
- Método `Main()`
- Inicialização do sistema
- Execução da tela principal
